# Aktieanalys – Pluggmaterial

Ett interaktivt pluggmaterial för att lära sig grunderna i aktieanalys.

## Innehåll
- De tre finansiella rapporterna
- Nyckeltal (P/E, EV/EBIT, ROE m.m.)
- Värdering (DCF, relativvärdering, margin of safety)
- Kvalitativ analys (moat, ledning, risker)
- Mentala modeller (Mr. Market, bekräftelsebias m.m.)
- Quiz med 10 frågor

## Publicera med GitHub Pages

1. Ladda upp filerna till ett nytt GitHub-repo
2. Gå till **Settings → Pages**
3. Under *Branch*, välj `main` och mappen `/ (root)`
4. Klicka **Save** – din sida är live på `dittnamn.github.io/reponamn`
